# Your Energy

Frontend project with an exercises catalog, filters, search, pagination and favorites.

## Live
- Demo: https://emmaloi.github.io/your-energy/

## Features
- Exercise categories and filters
- Exercise list with pagination
- Search by keyword
- Favorites (LocalStorage)
- Exercise details modal with rating

## Tech Stack
- Vite
- HTML5
- SCSS
- JavaScript (ES Modules)

## Scripts
```bash
npm install
npm run dev
npm run build
```
